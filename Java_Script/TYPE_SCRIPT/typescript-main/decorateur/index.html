Les Descripteurs

writable : il spécifie si la propriété peut être modifié après sa création. Si "writable" est "false", la propriété est considérée comme en lecteur seule

enumerable : il indique si la propriété sera listée lors du parcours de l'objet à l'aide d'une boucle "for ... in" ou "Object.keys"

configurable : il spécifie si la propriété peut être supprimée ultérieurement, et si ses descripteurs peuvent être modifiés
