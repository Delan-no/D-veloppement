Les variables

        
declaration

        
En php, toutes les variables commmencent par un "$". Toute déclaration php, se termine toujours par un point-virgule
    Exemple:
        $user="John";
        $num = 2

Convention

la convention de nommage des variables est la suivante :

1- Toutes les variables commencent forcément par un dollar
2- Le nom d'une variable doit être composé uniquement de charactères alpha-numérique et du tiret de 8(_)
3- Le nom d'une variable ne doit pas commencer par un chiffre

Les types de variable

En php le type d'une variable lui est assigné quand vous lui donnez une valeur. On utilise la fonction ` var_dump()` pour avoir les informations, concernant le type d'une variable.

    Exemple:
        $user;
        var_dump($user)

String

réprésente les chaînes de caractère. il se déclare avec soit un guillemet double("") ou un guillemet simple('').
La concaténation de deux chaines de caractères se fait avec un point (.) 

        $fname = "John";
        $lname = "Doe";
          echo $fname . " " . $lname; // John Doe

Number

répresente tous les nombres; entiers comme à virgules. on en distingue deux séparations:
1 - int: désignant les nombres entiers
2 - float: désignant les nombres flottants (nombres décimaux)

        $num1 = 3;
        $moyenne = 18.5;
          var_dump($num1); // int
          var_dump($moyenne); // float

Boolean

répresente les valeurs de vérités; soit "true" soit "false";

        $isAdmin = true;
        $isEven = false;
          var_dump($isAdmin) ;// bool

null

répresente l'absence de valeurs

        $data = null;
          var_dump($data); // null

Type conversion

pour convertir d'un type à un autre, il suffit de préciser l'opérateur de conversion explicite.

        $floatNum = 3.14;
        $intNum = (int)$floatNum; // on obtient un entier

En plus, nous pouvons utiliser les fonctions php natives telles que *intval()*, *floatval()*, *boolval()* ...

Conditions

Les conditions en PHP sont identiques au JS, et on y retrouve les mêmes opérateurs de comparaison tels que : <, <=, >, >=, !=, =, ==, !==

PHP-FIG

(PHP Framework Interop Group) est un groupe de travail ingormel composé de développeurs et de projets open source qui travaillent dans l'écosystème PHP.

L'objectif principal de PHP-FIG est de créer et de promouvoir des normes de codage et de conception communes pour les f rameworks et les bibliothèques PHP, afin de facilliter l'interopérzabilité entre différents projets.

En d'autres termes, PHP-FIG visse à étéblir des recommandations et des standards pour que les différents projets PHP puissent fonctionner ensemble de manière harmonieuse.

Le groupe a produit plusieurs PSR (PHP Standard Recommandations) qui sont des documents décrivant les normes de codage et de conception pour différents aspects du dévelopement PHP, tels que l'autoloading des classes, les interfaces HTTP...
